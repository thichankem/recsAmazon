Hệ gợi ý sử dụng dữ liệu amazon

1. Ý tưởng: áp dụng thuật toán collaboration filtering và content-based filtering để gợi ý sản phẩm phù hợp nhất cho khách hàng

2. datalink: https://amazon-reviews-2023.github.io/

3. cấu trúc:    

- data: dữ liệu training

- preprocessing: tiền xử lý các dữ liệu 

- model: nghiên cứu các thuật toán

- model_output: lưu các mô hình thuật toán

- test: kiểm thử online với web amazon thực tế dùng playwirght

- frontend: giả lập web amazon để kiểm thử offline

- evaluation: chấm điểm các bài test

4. Các công nghệ được sử dụng: 
Phát triển: anaconda, ngôn ngữ python, jupyter notebook, thư viện chủ yếu
